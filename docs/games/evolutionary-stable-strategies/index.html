Theory: Evolutionary stable strategies

As we saw earlier, one of the interpretations of mixed strategy Nash equlibrium is that it determines the fraction of population playing a certain pure action. Maynard Smith and Price (1973) used this interpretation to propose an evolutionary mechanism via which agents can change their behavior across generations. For a facinating (non-technical) introduction to this area, see Maynard Smith (1982) and Dawkins (1976).

The key idea in what is now called Evolutionary Game theory is that of evolutionary stable strategies, which we discuss here. For a detailed treatment of the subject, see Sandholm (2010).

To understand evolutionary stability, consider the following variant of the Hawk-Dove game.

\(\mathsf{D}\) \(\mathsf{H}\)
\(\mathsf{D}\) \(4\) \(4\) \(2\) \(8\)
\(\mathsf{H}\) \(8\) \(2\) \(1\) \(1\)

Note that this is a symmetric game. Now suppose that there is a large population comprising of 80% doves and 20% hawks. Thus, we are considering a population state \(α = (0.8, 0.2)\). We assume that there are repeated interactions between the agents in the population. At state \(α\) described above, a generic agent will encounter a dove with probability \(0.8\) and a hawk with probability \(0.2\). Thus, if this agent is a dove, his payoff is \[ U(\mathsf{D},α) = 0.8×4 + 0.2×2 = 3.6. \] If this agent is a hawk, then his payoff is \[ U(\mathsf{H},α) = 0.8×8 + 0.2×1 = 6.6 \]

Maynard Smith (1982) argues that in this situation, the population of Hawks have an evolutionary advtange. So, over time, the fraction of hawks will increase. Thus, the population state \(α^*\) where no trait has an advantage is \[ U(\mathsf{D},α^*) = U(\mathsf{H},α^*), \] which is precisely the Nash equilibrium of the game.

1 Evolutionary stable strategies

The key difference in this interpretation from that of Nash equilibrium is hwo we determine if an equilibrium is sustainable. In particular, consider a symmetric two player game \(\mathscr{G} = \langle 2, (\ALPHABET B, \ALPHABET B), (u_1, u_2) \rangle\) where \(u(a_1, a_2) = u(a_2, a_1)\). We will denote the payoff function as \(u \colon \ALPHABET B × \ALPHABET B \to \reals\) where \[ u(a,b) = u_1(a,b) = u_2(b,a). \] Now consider a symmetric mixed strategy \(α^*\), i.e., we have a large population where \(α^*(a_i)\) is the fraction of the population that plays strategy \(a_i\) and so on.

Suppose a fraction \(ε\) of the population mutates and starts playing another strategy \(α\). Thuis, the new state of the population is \[ \bar α = (1-ε)α^* + ε α. \] The expected payoff of “normal” player (playing against the new mixed population) is \[ U(α^*, \bar α) = (1-ε)U(α^*, α^*) + ε U (α^*, α). \] Similarly, the payoff of a mutated player (playing against the new mixed population) is \[ U(α, \bar α) = (1-ε)U(α, α^*) + ε U (α, α). \]

Definition

A population state \(α^*\) is said to be an evolutionary stable strategy (ESS) if for all mutations \(α \in Δ(\ALPHABET B)\), \(α \neq α^*\), there exists a \(ε_\circ = ε_\circ(α) > 0\) such that for all \(ε \in (0, ε_\circ)\), and \(\bar α_{ε} = (1-ε)α^* + ε α\), we have \[ U(α^*, \bar α) > U(α, \bar α) \] i.e., \[\begin{equation}\label{eq:ESS} (1-ε)U(α^*, α^*) + ε U (α^*, α). > (1-ε)U(α, α^*) + ε U (α, α). \end{equation}\]

Remark

If we take the limit of \(ε \to 0\) in the definition of ESS, we get that an ESS has to be a (symmetric) Nash equilibrium. The next example illustrates that the reverse is not always true.

1.1 An example to illustrate that all Nash equilibria are not ESS

Consider a variation of the Hawk-Dove game

\(\mathsf{D}\) \(\mathsf{H}\)
\(\mathsf{D}\) \(4\) \(4\) \(2\) \(2\)
\(\mathsf{H}\) \(2\) \(2\) \(2\) \(2\)

This game has two equilibria in pure strategies \((\mathsf{D}, \mathsf{D})\) and \((\mathsf{H}, \mathsf{H})\). We now verify if each of them is an ESS or not.

Verification for \((\mathsf{D}, \mathsf{D})\)

In this case \(α^* = (1,0)\). Consider any mutation \(α = (p, 1-p)\) with \(p \neq 1\). Then \[\begin{align*} U(α^*, α^*) &= 4 \\ U(α^*, α) &= 4p + 2(1-p) = 2p + 2 \\ U(α, α^*) &= 4p + 2(1-p) = 2p + 2 \\ U(α,α) &= 4p^2 + 2(1-p^2) = 2p^2 + 2 \end{align*}\] Thus, \[ U(α^*, \bar α) = (1-ε)4 + ε(2p+2) = 4 - 2ε (1-p) \] and \[ U(α, \bar α) = (1-ε)(2p+2) + ε(2p^2 + 2) = (2p+2) - 2pε(1-p) \] Now consider \[\begin{align*} U(α^*, \bar α) - U(α, \bar α) &= 2(1-p) - 2ε(1-p)^2 \\ &= 2(1-p)[ 1 - ε(1-p) ] \end{align*}\] Since \(p \neq 1\) and \(ε(1-p) < 1\), we get that \(U(α^*, \bar α) > U(α, \bar α)\). Hence \(α^*\) is an ESS.

Verification for \((\mathsf{H}, \mathsf{H})\)

In this case \(α^* = (0,1)\). Consider any mutation \(α = (p, 1-p)\) with \(p \neq 0\). Then \[\begin{align*} U(α^*, α^*) &= 2 \\ U(α^*, α) &= 4p + 2(1-p) = 2 \\ U(α, α^*) &= 4p + 2(1-p) = 2 \\ U(α,α) &= 4p^2 + 2(1-p^2) = 2p^2 + 2 \end{align*}\] Thus, \[ U(α^*, \bar α) = (1-ε)2 + ε2 = 2 \] and \[ U(α, \bar α) = (1-ε)2 + ε(2p^2 + 2) = 2 + 2εp^2 \] Now consider \[\begin{align*} U(α^*, \bar α) - U(α, \bar α) &= -2εp^2 < 0 \end{align*}\] since \(p \neq 0\). Hence \(α^*\) is not an ESS.

2 An easier to verify characterization of ESS

Theorem 1

A strategy \(α^*\) is ESS iff for every \(α \neq α^*\), one of the following conditions is true:

  1. \(U(α^*, α^*) > U(α, α^*)\)
  2. \(U(α^*, α^*) = U(α, α^*)\) and \(U(α^*, α) > U(α,α)\).

Condition 1 states that if a mutation deviates from \(α^*\), it will lose in its encounters with the normal population. Condition 2 states that if the payoff that a mutation receives from encountering a normal individual is equal to the pay off a normal individual receives when encountering a normal individual, that mutation will receive a smaller payoff if it encounters the same mutation that a normal individual would in the encounter.

Proof that ESS implies conditions (1) and (2)

Let \(α^*\) be an ESS. As argued earlier, this implies that \(α^*\) is also a Nash equilibrium. Thus, \[ U(α^*, α^*) \ge U(α, α^*), \quad \forall α \in Δ(\ALPHABET B). \] Now suppose that neither conditions (1) nor (2) hold. Then, there must exist an \(α \neq α^*\) such that \[ U(α^*, α^*) = U(α, α^*) \quad\text{and}\quad U(α^*, α) \le U(α,α). \] This implies that for any \(ε \in (0,1)\), \[ (1-ε)U(α^*,α^*) + εU(α^*, α) \le (1-ε)U(α, α^*) + εU(α, α), \] which contradicts the fact that \(α^*\) is ESS.

Proof that conditions (1) and (2) imply ESS

Suppose a mutation \(α\) satisfies Condition (1). Let \(D = U(α^*,α^*) - U(α,α^*)\). Then, \[\begin{align*} U(α^*, \bar α) - U(α, \bar α) &= (1-ε)\bigl[ U(α^*, α^*) - U(α, α^*) \bigr] - ε \bigl[ U(α, α) - U(α^*, α) \bigr] \\ &\ge (1 - ε) D - ε M \end{align*}\] where \(M = \SPAN(u) := \max_{a,b} u(a,b) - \min_{a,b} u(a,b)\). Now, for any \(ε < D/(D + M)\), we have that \(U(α^*, \bar α) - U(α, \bar α) > 0\). Thus, \(α^*\) is an ESS.

Now suppose that a mutation \(α\) satisfies condition (2). Then, \[\begin{align*} U(α^*, \bar α) - U(α, \bar α) &= (1-ε)\underbrace{\bigl[ U(α^*, α^*) - U(α, α^*) \bigr]}_{=0} + ε \underbrace{\bigl[ U(α^*, α) - U(α, α) \bigr]}_{\ge 0} \\ &> 0, \quad \forall ε \in (0,1). \end{align*}\] Thus, \(α^*\) is an ESS

As an example, let’s revist the game considered above.

\(\mathsf{D}\) \(\mathsf{H}\)
\(\mathsf{D}\) \(4\) \(4\) \(2\) \(8\)
\(\mathsf{H}\) \(8\) \(2\) \(1\) \(1\)

We can verity that \(α^* = (\frac 15, \frac 45)\) is the unique Nash equilibrium. Now consider a mutation \(α = (p, 1-p)\). To check if \(α^*\) is ESS, we first check condition 1 of Theorem 1. Since \(α^*\) is a Nash equilibrium with support \(\{\mathsf{H},\mathsf{D}\}\), we must have \[ U(\mathsf{H},α^*) = U(\mathsf{D},α^*). \] Therefore, for any strategy \(α\) \[ U(α, α^*) = U(α^*,α^*). \] So condition (1) is not satisfied. Condition (2) will be statisfied if \[\begin{align*} && U(α^*, α) &> U(α,α) \\ &\iff& 4 \cdot \frac p5 + 2 \cdot \frac {(1-p)}{5} + 5 \cdot \frac{4p}{5} + 1 \cdot \frac{4(1-p)}{5} &> 4p^2 + 2p(1-p) + 8(1-p)p + (1-p)^2 \\ &\iff& (5p-1)^2 &> 0 \end{align*}\] which holds for all \(p \neq \frac 15\). Thus, \(α^*\) is an ESS.

3 ESS for arbitrary two player symmetric games

Consider a symmetric game of the form

\(\mathsf{T}\) \(\mathsf{B}\)
\(\mathsf{T}\) \(a\) \(a\) \(b\) \(c\)
\(\mathsf{B}\) \(c\) \(b\) \(d\) \(d\)

We will assume that \(b \neq d\) and \(a \neq c\). Let \(α^* = (p, 1-p)\) be a symmetric (strictly) mixed strategy equilibrium. By the indifference principle, we must have \[ a p + b(1-p) = cp + d(1-p) \implies p = \frac{b-d}{(b+c) - (a+d)} \] This \(p\) is a valid probability distribution if:

  • Case 1: \(\text{num} > 0\), \(\text{den} > 0\), and \(\text{num} < \text{den}\).

    Thus, $b > d$ and $c > a$.

  • Case 2: \(\text{num} < 0\), \(\text{den} < 0\), and \(|\text{num}| < |\text{den}|\).

    Thus, $b < d$ and $c < a$.
Proposition 1

A symmetric mixed strategy in a symmetric two player two action game is an ESS if \(b > d\) and \(c > a\).

Remark

When \(b > d\) and \(c > a\), the anti-symmetric strategies with payoffs \((b,c)\) and \((c,a)\) are Nash equilibria.

Proof

Since \(p \in (0,1)\), for any other strategy \(α = (q, 1-q)\), we will have \(U(α,α^*) = U(α^*, α^*)\). So, we need to check condition (2) of 2

\[\begin{align*} U(α,α^*), U(α,α) &= p\bigl[ qa + (1-q) b \bigr] + (1-p) \bigl[ q c + (1-q) d \bigr] \\ &- q\bigl[ qa + (1-q) b \bigr] - (1-q) \bigl[ q c + (1-q) d \bigr] \\ &= (p-q) \bigl[ qa + (1-q) b - q c - (1-q) d \bigr] \\ &= (p-q) \bigl[ (b-d) - q( (b+d) - (a+d) ) \bigr] \\ &= (p-q)^2 \bigl[ (b+c) - (a+d) \bigr] \\ &> 0. \end{align*}\] Thus, when \(b > d\) and \(c > a\), the mixed strategy equilibrium is an ESS.

4 Examples

4.1 Robotics

Consider the robotics example where \(r_b/2 < r_s < r_b\).

\(\mathsf{B}\) \(\mathsf{S}\)
\(\mathsf{B}\) \(\tfrac {1}{2}r_b\) \(\tfrac {1}{2}r_b\) \(r_b\) \(r_s\)
\(\mathsf{S}\) \(r_s\) \(r_b\) \(\tfrac {1}{2}r_s\) \(\tfrac {1}{2}r_s\)

In the notes on Nash equilibrium, we computed a symmetric mixed strategy Nash equilibrium with \[ p = \frac{2r_b - r_s}{r_b + r_s}. \] Since \(r_b > \frac{1}{2} r_s\) and \(r_s > \frac{1}{2} r_b\), this is also ESS.

4.2 Medium access control

Consider the example of medium access control where \(c < v\).

\(\mathsf{T}\) \(\mathsf{D}\)
\(\mathsf{T}\) \(-c\) \(-c\) \(v-c\) \(0\)
\(\mathsf{D}\) \(0\) \(v-c\) \(0\) \(0\)

The symmetric mixed strategy Nash equilibrium in this case is \[ p = 1 - \frac{c}{v} \]

Since \(v - c > 0\) and \(0 > -c\), this is also an ESS.

4.3 Medium access control with lost opportunity cost

Consider the example of medium access control with lost opportunity cost where \(c < v\).

\(\mathsf{T}\) \(\mathsf{D}\)
\(\mathsf{T}\) \(-c\) \(-c\) \(v-c\) \(0\)
\(\mathsf{D}\) \(0\) \(v-c\) \(-d\) \(-d\)

The symmetric mixed strategy Nash equilibrium in this case is \[ p = 1 - \frac{c}{v+d} \]

Since \(v - c > -d\) and \(0 > -c\), this is also an ESS.
